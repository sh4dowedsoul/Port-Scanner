# Port-Scanner
# 🔐 Simple Port Scanner (Python)

## 📌 About the Project

This is a basic port scanner made using Python. It checks open ports on a target system like a website or IP address.

## ⚙️ Features

* Scan open ports of a target
* Works with IP address or domain name
* Shows open ports in terminal
* Simple and easy to understand code

## 🛠️ Technologies Used

* Python
* Socket library

## ▶️ How to Run

1. Make sure Python is installed
2. Open terminal in project folder
3. Run the command:

```bash
py port_scanner.py google.com
```

or

```bash
py port_scanner.py 127.0.0.1
```
